Built for people who get things done

A task manager that doesn't just organize your work — it does it.

Daily task board

Organize your work by day with an accordion view spanning the last week. Each task expands into a full markdown spec.

AI agent in a sandbox

Your agent runs in an isolated VM with real shell access, file operations, and web search — safely contained.

Chat interface

Talk to your agent naturally. Delegate tasks, ask questions, or have it write code — powered by Claude.

Artifacts panel

The agent saves reusable files — scripts, configs, notes — to a shared artifacts directory you can browse.

Local-first

Everything runs on your machine. Your data stays in a local directory, mounted directly into the sandbox.

Light & dark themes

Follows your system preference or toggle manually. Clean, minimal interface inspired by Apple's design language.

Three steps to delegation

From task to result in minutes, not hours.

1

Add a task

Create a task on today's board. Write a markdown spec with as much or as little detail as you want.

2

Delegate to the agent

Open the chat and tell the agent what to do. It spins up a sandbox VM and gets to work.

3

Review the results

The agent's output appears as artifacts — files, code, research — ready for you to review and use.
